📡 API Endpoints

POST /api/v1/remove/image

Remove watermark from an image. Returns processed image URL.

Request

  • image (file) - Image file (JPG, PNG, WebP, max 20MB)

Response

{
  "success": true,
  "message": "Image processed successfully",
  "downloadUrl": "/api/v1/download/filename.png",
  "filename": "processed_image.png",
  "meta": {
    "originalName": "original.jpg",
    "size": 1234567,
    "processedAt": "2026-07-06T10:00:00.000Z"
  }
}
POST /api/v1/remove/video

Remove watermark from a video. Returns job ID for async processing.

Request

  • video (file) - Video file (MP4, MOV, MKV, AVI, max 500MB)

Response

{
  "success": true,
  "message": "Video processing started",
  "jobId": "abc123-def456-ghi789",
  "statusUrl": "/api/v1/status/abc123-def456-ghi789",
  "estimatedTime": "2-5 minutes depending on video length"
}
GET /api/v1/status/:jobId

Check the status of a video processing job.

Response (Processing)

{
  "jobId": "abc123-def456-ghi789",
  "status": "processing",
  "type": "video",
  "createdAt": "2026-07-06T10:00:00.000Z",
  "completedAt": null,
  "progress": 45
}

Response (Completed)

{
  "jobId": "abc123-def456-ghi789",
  "status": "completed",
  "type": "video",
  "createdAt": "2026-07-06T10:00:00.000Z",
  "completedAt": "2026-07-06T10:05:00.000Z",
  "downloadUrl": "/api/v1/download/processed_video.mp4",
  "filename": "processed_video.mp4"
}
GET /api/v1/download/:filename

Download a processed file.

Example

curl -O http://localhost:3000/api/v1/download/processed_video.mp4 \
  -H "X-API-Key: demo-key-12345"
